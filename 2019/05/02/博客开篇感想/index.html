博客开篇感想

发表于 2019-05-02 | | 阅读次数:

一曲阳关奏罢，送走了那无计可消除的离愁别绪，留下了伊人那隐约可见的倩影。
人有悲欢离合，月有阴晴圆缺，此事古难全。一天天，一月月，一年年，道不尽那绵绵无尽的相思与眷恋，诉不完那历久弥坚的海誓山盟。此情可待成追忆，只是当时已惘然。
人生苦短，依稀可见往日伊人笑靥如春花般悄然盛放。曾经沧海不可追，剩下的零丁散碎记忆也只余终成微尘一途可走。
无可奈何花落去，草木荣枯自有时。容颜易改，伊人难再。君不见，那像追寻着丁香一样的结着愁怨的姑娘，依然在寂寥的雨巷中哀怨彷徨。
今人不见古时月，汉时阴山，今夕烟火，恋之一字，伤人太多。当爱已成过往，昔时欢愉，今宵夜寐，俱为他人做嫁。更那堪，当年芳华，难逃岁月消磨，无尽残荷，寂寞沙洲冷。
问世间情为何物，梁祝羽化已成蝶。从今而后，清酒金樽，寒沙冷月，都化作一缕愁思，埋在心底深处，难再翻起。岳鄂王诗云：“欲将心事付瑶琴，知音少，弦断有谁听？”此句极妙。
有人曾解读商末帝辛与西周幽王，牧野悲歌，连城烽火，荒淫无度的背后到底还是高处不胜寒的孤寂与苦涩。
而身染红尘的我们，或许也正如《冰菓》第二刊序言里说的那样，终将成为岁月和历史的牺牲。每念及此，心中总会感到一种无可排解的悲哀。
今时今日，尚算年轻的我，不想就这样顺着既定的路线碌碌无为地在自己不感兴趣的行业中蹉跎光阴，消磨斗志。这个博客，就是我对过往与将来的反抗。
从今而后，前路莫问，我只努力在当下，活在当下。
